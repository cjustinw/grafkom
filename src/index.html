Help

  • Pilih terlebih dahulu Mode
  • Mode Draw (gambar) : Pilih Shapes (Line/Square/Rectangle/Polygon), lalu mulai gambar di kanvas. Tekan q untuk cancel. Untuk poligon, tekan enter saat selesai menggambar.
  • Mode Move (geser garis) : Geser garis yang diinginkan pada kanvas.
  • Mode Scale (ubah ukuran garis/panjang persegi) : Atur Scale yang diharapkan lalu klik objek target.
  • Mode Color (ubah warna) : Pilih Color yang diharapkan lalu klik objek target.
  • Untuk menyimpan pengerjaan, tuliskan nama file lalu tekan Export.
  • Untuk membuka file yang telah disimpan, pilih file lalu tekan Import.
